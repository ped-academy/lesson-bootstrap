# PED Academy
## Web Developer #LEZ5
### Flexbox & Bootstrap

Questo repository contiene codice HTML e CSS d'esempio con gli esercizi visti a lezione.

Clonate il repository nella cartella `htdocs` del vostro web server con il seguente comando:

```
git clone https://github.com/ped-academy/lesson-bootstrap.git
```

Visitate la pagina [http://localhost/lesson-bootstrap/exercises](http://localhost/lesson-bootstrap/exercises) per visualizzare la lista degli esercizi
